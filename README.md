# Smart Attendance & Engagement Monitoring System

## Project Description
The Smart Attendance & Engagement Monitoring System is a web-based application designed to automate student attendance and analyze classroom engagement. It replaces traditional manual attendance methods with a digital solution.

## Problem Statement
Manual attendance systems are time-consuming, prone to proxy attendance, and do not provide insights into student engagement. Institutions lack a unified platform to track both attendance and participation effectively.

## Proposed Solution
This project provides a digital attendance system combined with engagement monitoring through quizzes and participation tracking. Data is stored securely and presented using analytics dashboards to support informed academic decisions.

## Technology Stack
- Frontend: React
- Backend: Node.js
- Database: MongoDB
- Tools: GitHub, VS Code

## Key Features
- Role-based login (Student, Faculty, Admin)
- Digital attendance marking
- Engagement tracking
- Attendance analytics dashboard
- Secure data storage

## Project Workflow
1. User authentication
2. Role-based dashboard
3. Attendance marking
4. Engagement activities
5. Data storage
6. Analytics and reporting

## Future Scope
- Face recognition attendance
- Mobile app integration
- AI-based engagement prediction
